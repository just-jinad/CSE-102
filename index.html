Arrays are methods used to store information and can be looped through to be displayed back to the user, or fetch back specific information when needed.

2. Array.splice: array.splice is used to add or remove elements from an array.

3. Array.pop: array.pop is used to remove elements from the back of an array.

4. Arrary.slice: is used to remove elements from an array.

5. Array.filter: is used to return elements that matches the condition given.
